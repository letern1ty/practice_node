## 二、 初始化项目
####    1. vue初始化
        vue create demo 
* 此处选用vue3 来创建项目 
* 主要的功能技术栈有： vite + element—plus
####  2. 后台项目初始化（nodejs）
全局安装 express-generator
`npm install express-generator -g`
使用 express 创建项目 
`express --view=ejs server `
启动后台项目
`http://localhost:3000`
